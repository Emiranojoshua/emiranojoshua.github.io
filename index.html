What is HTML?

Javascript is dynamic, intepreted and weekly typed programming language which was built with the purpose of running in host environment (e.g. inside of a webpage in the browser).

It's part of pretty much any page you visit in the web as it's the only "thing" in web development that allows you ("the developer") to change something on the page at runtime. Basically every time something opens, closes, moves or chnges on a webpage, Javascript will be involved.

There is also a lot of webpages which are almost entirely powered by Javascript (i.e. where a lot of the rendering work happens in the browser), those pages typically use frameworks like Angular, but that a topic we'll cover later in this course)
